# API Reference

> JSON-RPC method and event reference. _(To be written when the protocol is locked in Phase 4.)_

This section will contain the full reference for the Synaptic IPC protocol:

- **Methods** — every JSON-RPC method, its parameters, return type, and examples.
- **Events** — every server-pushed event, its payload, and when it fires.
- **Types** — every shared type, its schema, and example values.
- **Errors** — every error code, its meaning, and how to handle it.

---

These will be written in **Phase 4**, after the IPC protocol is locked for v0.1.0. The wire format is JSON-RPC 2.0 with one extension for streaming (see [../architecture/09-ipc.md](../architecture/09-ipc.md)).
