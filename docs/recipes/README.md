# Recipes

> Worked examples: how to do specific things with Condura. _(To be written in later phases.)_

This section will contain step-by-step recipes for common use cases:

- **Research Agent** — "Research X and write me a report."
- **Dev Helper** — "Help me with my codebase, run tests, fix bugs."
- **Safari Automation** — "Open this URL, fill in this form, click submit."
- **Scheduled Tasks** — "Every Monday at 9am, summarize my unread emails."
- **Email Triage** — "Read my inbox, categorize, draft replies."
- **Calendar Management** — "Find a 1-hour slot for a meeting with X next week."
- **File Organization** — "Sort my Downloads folder by type and date."
- **Trip Planning** — "Plan a 7-day trip to Tokyo, including flights, hotel, and itinerary."
- **Code Review** — "Review this PR and suggest improvements."
- **Browser Tasks** — "Find the cheapest flight, the best hotel, and the top-rated restaurant."

---

These will be written in **Phase 6 or later**, after the sub-agents and skills are mature.
